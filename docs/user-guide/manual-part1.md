## 第一章：系统概述

### 1.1 产品介绍

**千牛云呼叫中心系统是什么？**

简单来说，千牛云呼叫中心系统就是一套专门用来**接打电话、管理客户、处理问题**的软件平台。

您可以把它想象成一个"超级电话总机"——客户打进来的电话，会自动分配给合适的客服人员（也就是"座席"）；座席也可以主动给客户打电话。与此同时，系统还会自动记录每一通电话、每一次服务，方便管理人员查看和分析。

**这个系统能做什么？**

| 功能 | 说明 |
|------|------|
| 接听来电 | 客户打进来的电话，系统自动分配给空闲的座席接听 |
| 主动拨出 | 座席可以主动给客户打电话 |
| 客户信息管理 | 记录客户的姓名、电话、历史服务记录等信息 |
| 工单管理 | 把客户的问题记录成"工单"，跟踪处理进度直到解决 |
| 通话录音 | 自动录制每一通电话，方便事后回听和质量检查 |
| 数据统计 | 统计接听量、通话时长、服务质量等数据 |
| 质量检查 | 质检员可以回听录音，评估座席的服务质量 |

**谁会用到这个系统？**

- 客服座席：每天用它接听和拨打客户电话
- 座席主管：用它监控团队工作状态，查看数据报表
- 管理员：用它配置系统参数，管理账号权限
- 质检员：用它回听录音，评分打分

### 1.2 系统角色说明

系统中有四种不同的角色，每种角色能看到和操作的功能不同：

#### 🧑‍💼 管理员（Administrator）

管理员是权限最高的角色，负责整个系统的配置和管理。

**能做什么：**
- 创建和管理所有用户账号（新增座席、修改密码等）
- 配置系统参数（如电话线路、IVR语音菜单等）
- 管理技能组（把座席分配到不同的服务组）
- 查看所有数据报表和统计信息
- 管理客户数据
- 查看所有通话录音

⚠️ **注意：** 管理员账号权限极大，请妥善保管密码，不要随意告诉他人。

#### 👨‍💼 座席主管（Supervisor）

座席主管负责管理和监督一组座席的日常工作。

**能做什么：**
- 实时查看所管辖座席的工作状态（谁在通话、谁在休息）
- 监听座席的通话（座席不知道被监听）
- 强插通话（紧急情况下加入座席正在进行的通话）
- 查看本组的数据报表
- 处理座席提交的工单
- 给座席发送消息提醒

#### 🎧 座席（Agent）

座席是直接与客户通话的一线人员，也是使用系统最频繁的角色。

**能做什么：**
- 接听客户来电
- 主动拨打客户电话
- 查看和编辑客户信息
- 创建和处理工单
- 查看自己的通话记录
- 填写通话小结

#### 🔍 质检员（Quality Inspector）

质检员负责检查座席的服务质量。

**能做什么：**
- 回听所有座席的通话录音
- 对通话进行评分和打分
- 填写质检报告
- 查看质检统计数据

### 1.3 使用前准备

在开始使用系统之前，请确认您的电脑和网络环境满足以下要求：

#### 💻 电脑要求

| 项目 | 最低要求 | 推荐配置 |
|------|----------|----------|
| 操作系统 | Windows 7 / macOS 10.12 | Windows 10 / macOS 12 以上 |
| 内存（RAM） | 4GB | 8GB 以上 |
| 处理器 | 双核 1.8GHz | 四核 2.4GHz 以上 |
| 硬盘剩余空间 | 500MB | 2GB 以上 |
| 显示器分辨率 | 1280×768 | 1920×1080（推荐） |

💡 **提示：** 如果您的电脑运行缓慢，可能会影响通话质量，建议关闭不必要的程序后再使用本系统。

#### 🌐 浏览器要求

本系统通过网页浏览器使用，**推荐使用以下浏览器**：

| 浏览器 | 版本要求 | 推荐程度 |
|--------|----------|----------|
| Google Chrome（谷歌浏览器） | 80 版本以上 | ⭐⭐⭐⭐⭐ 强烈推荐 |
| Microsoft Edge（微软 Edge） | 80 版本以上 | ⭐⭐⭐⭐ 推荐 |
| Firefox（火狐浏览器） | 75 版本以上 | ⭐⭐⭐ 可用 |

⚠️ **注意：** 请**不要**使用 Internet Explorer（IE浏览器）或 360浏览器的兼容模式，这些浏览器可能导致系统无法正常使用。

**如何查看浏览器版本？**
1. 打开 Chrome 浏览器
2. 点击右上角的三个点（⋮）
3. 选择"帮助"→"关于 Google Chrome"
4. 即可看到当前版本号

#### 🌍 网络要求

| 项目 | 要求 |
|------|------|
| 网络类型 | 有线网络（推荐）或 WiFi |
| 网络带宽 | 上传和下载各不低于 2Mbps |
| 网络延迟 | 低于 100ms（越低越好） |
| 网络稳定性 | 不能频繁断线 |

⚠️ **注意：** 通话质量对网络要求较高。如果您使用 WiFi，请尽量靠近路由器，避免信号弱导致通话断断续续。**强烈建议使用有线网络连接。**

💡 **提示：** 如何测试网络速度？打开浏览器，访问 `www.speedtest.net`，点击"开始测试"即可查看您的网络速度。

#### 🎧 耳机要求

接打电话需要使用耳机（带麦克风），推荐配置如下：

| 项目 | 说明 |
|------|------|
| 耳机类型 | 头戴式耳机（带麦克风）或入耳式耳机（带麦克风） |
| 接口类型 | USB 接口或 3.5mm 音频接口均可 |
| 降噪功能 | 推荐使用带主动降噪功能的耳机，减少背景噪音 |
| 麦克风 | 必须有麦克风，否则无法通话 |

⚠️ **注意：** 请不要使用电脑内置麦克风和扬声器进行通话，这样客户会听到回声，影响通话质量。

### 1.4 常用术语解释

如果您是第一次接触呼叫中心，下面这些词可能会让您感到陌生。别担心，我们用大白话来解释：

| 术语 | 大白话解释 |
|------|-----------|
| **座席** | 就是客服人员，负责接打电话的员工。您自己就是一名"座席"。 |
| **呼入** | 客户主动打电话进来，系统接收到这个电话，叫做"呼入"。 |
| **呼出** | 座席主动给客户打电话出去，叫做"呼出"。 |
| **IVR** | 就是您打客服电话时听到的"请按1查询余额，请按2转人工"那种自动语音菜单。IVR 是"交互式语音应答"的缩写。 |
| **工单** | 把客户的问题记录下来，形成一张"任务单"，跟踪这个问题从发现到解决的全过程，就叫工单。 |
| **技能组** | 把座席按照专业技能分成不同的组，比如"售前咨询组"、"售后服务组"、"投诉处理组"等。客户来电时，系统会把电话分配给对应技能组的座席。 |
| **排队** | 当所有座席都在通话中，新来的客户电话会在系统里等待，这就叫"排队"。 |
| **转接** | 把正在进行的通话转给另一个座席或部门，就像把电话"传"给别人。 |
| **保持** | 通话中暂时让客户等待（客户会听到等待音乐），座席可以去查资料或请示主管，叫做"保持"。 |
| **静音** | 座席把自己的麦克风关掉，客户听不到座席说话，但座席还能听到客户，叫做"静音"。 |
| **三方通话** | 座席、客户、第三方（比如另一个座席或专家）三个人同时在一个电话里通话。 |
| **通话小结** | 每次通话结束后，座席填写的一段简短记录，说明这次通话的内容和处理结果。 |
| **录音** | 系统自动把每次通话录制下来，保存成音频文件，方便事后回听。 |
| **软电话** | 不是实体电话机，而是安装在电脑上、通过网络打电话的软件。本系统内置了软电话功能。 |
| **在线状态** | 座席当前的工作状态，比如"空闲"（可以接电话）、"通话中"（正在打电话）等。 |
| **弹屏** | 来电时，系统自动弹出客户信息窗口，让座席在接听前就能看到客户资料，叫做"弹屏"。 |
| **质检** | 质量检查的简称，质检员回听录音、评估座席服务质量的工作。 |
| **满意度** | 通话结束后，系统自动给客户发送评价邀请，客户对本次服务打分，叫做"满意度"。 |

## 第二章：登录与退出

### 2.1 如何打开系统

系统是通过网页浏览器使用的，不需要安装任何软件，只需要打开浏览器输入网址即可。

**详细步骤：**

1. 找到桌面上的 **Google Chrome** 图标（一个彩色圆形图标），双击打开浏览器。

2. 浏览器打开后，您会看到一个地址栏（在页面顶部，显示网址的长条输入框）。

3. 用鼠标点击地址栏，地址栏里的内容会变成蓝色（被选中状态）。

4. 直接输入系统网址（网址由您的管理员提供，格式类似：`https://callcenter.yourcompany.com`）。

5. 输入完成后，按键盘上的 **Enter（回车）** 键。

6. 等待几秒钟，页面加载完成后，您会看到千牛云呼叫中心系统的登录页面。

💡 **提示：** 为了方便以后快速打开系统，您可以把这个网址保存为书签：
- 在 Chrome 中，按 **Ctrl + D**（Mac 用户按 **Command + D**）即可收藏当前页面。
- 下次只需点击书签栏中的收藏，就能快速打开系统。

⚠️ **注意：** 请确认您输入的网址完全正确，特别注意不要多打或少打字母。如果不确定网址，请联系您的管理员确认。

### 2.3 登录操作步骤

**第一步：输入账号**

1. 用鼠标点击"账号"输入框（框内会出现闪烁的光标）。
2. 用键盘输入您的账号。账号通常是您的工号（如：`A001`）或手机号码，具体格式请询问您的管理员。
3. 输入完成后，可以看到输入框中显示了您输入的账号。

**第二步：输入密码**

1. 用鼠标点击"密码"输入框。
2. 用键盘输入您的密码。注意：密码输入时显示为圆点（●●●●），这是正常的，是为了保护您的密码安全。
3. 如果想确认密码是否输入正确，可以点击密码框右侧的**眼睛图标**（👁），密码会临时显示为明文，确认后再次点击眼睛图标隐藏。

**第三步：输入验证码（如果有）**

1. 查看验证码图片中显示的字符（通常是4-6位字母和数字的组合）。
2. 点击验证码输入框，输入您看到的字符。
3. 注意：验证码通常**不区分大小写**，但请尽量按照图片中的样式输入。
4. 如果验证码看不清楚，点击验证码图片可以刷新，获取新的验证码。

**第四步：点击登录**

1. 确认账号、密码、验证码都已正确输入后，点击蓝色的**"登录"**按钮。
2. 系统会显示"登录中..."的提示，请稍等几秒钟。
3. 登录成功后，页面会自动跳转到系统主界面（工作台）。

💡 **提示：** 输入完密码后，也可以直接按键盘上的 **Enter（回车）** 键来登录，不必用鼠标点击登录按钮。

### 2.4 登录失败怎么办

如果登录时出现错误提示，请根据提示信息对照下表处理：

| 错误提示 | 原因 | 解决方法 |
|----------|------|----------|
| "账号或密码错误" | 账号或密码输入有误 | 仔细检查账号和密码，注意大小写，重新输入 |
| "账号不存在" | 输入的账号在系统中没有 | 确认账号是否正确，联系管理员确认 |
| "密码错误，还剩X次机会" | 密码输入错误，有次数限制 | 仔细核对密码后重试，避免继续错误导致账号锁定 |
| "账号已被锁定" | 密码错误次数过多，账号被锁 | 联系管理员解锁账号 |
| "账号已禁用" | 账号被管理员停用 | 联系管理员确认账号状态 |
| "验证码错误" | 验证码输入有误 | 重新查看验证码图片，或点击刷新后重新输入 |
| "验证码已过期" | 验证码超时未使用 | 点击验证码图片刷新，重新输入新的验证码 |
| 页面无响应或空白 | 网络问题或浏览器问题 | 检查网络连接，刷新页面（按 F5 键），或换一个浏览器尝试 |

⚠️ **注意：** 如果连续输错密码超过5次，账号可能会被系统自动锁定，需要联系管理员解锁。请不要随意猜测密码。

**常见问题排查：**

- **账号输入错误：** 检查是否多输了空格，账号通常不包含空格。
- **密码大小写：** 密码区分大小写，请确认键盘的大写锁定键（Caps Lock）是否开启。键盘上的 Caps Lock 指示灯亮起表示大写锁定已开启。
- **中文输入法：** 确认输入账号和密码时，输入法是否切换到了英文模式。如果输入法显示中文，请按 **Shift** 键或点击输入法图标切换到英文。

### 2.5 首次登录如何修改密码

首次登录系统时，管理员会给您一个初始密码（通常是简单的数字组合，如 `123456`）。为了账号安全，**请务必在首次登录后立即修改密码**。

**修改密码步骤：**

1. 成功登录系统后，找到页面**右上角**的用户头像或您的姓名。
2. 用鼠标点击头像或姓名，会弹出一个下拉菜单。
3. 在下拉菜单中，找到并点击**"修改密码"**或**"个人设置"**选项。
4. 页面会弹出"修改密码"对话框，其中包含以下输入框：

| 输入框名称 | 说明 |
|------------|------|
| 当前密码（旧密码） | 输入您现在使用的密码（即管理员给您的初始密码） |
| 新密码 | 输入您想要设置的新密码 |
| 确认新密码 | 再次输入新密码，确保两次输入一致 |

5. 依次填写三个输入框。
6. 点击**"确认修改"**或**"保存"**按钮。
7. 系统提示"密码修改成功"后，您需要用新密码重新登录。

**密码设置要求：**

| 要求 | 说明 |
|------|------|
| 长度 | 通常要求 8-20 位字符 |
| 复杂度 | 建议包含大写字母、小写字母、数字、特殊符号（如 `@#$%`）中的至少两种 |
| 禁止使用 | 不能使用过于简单的密码，如 `123456`、`password`、`abcdef` |
| 不能与旧密码相同 | 新密码不能与当前密码完全相同 |

💡 **提示：** 好密码的设置技巧：可以用一句话的首字母加上数字和符号，例如"我2024年加入千牛云"可以设置为 `W2024jQNY!`，这样的密码既好记又安全。

⚠️ **注意：** 请不要把密码告诉任何人，包括您的主管和管理员。系统管理员不需要知道您的密码就能帮您解决问题。

### 2.7 忘记密码怎么办

💡 **提示：** 为了避免忘记密码的情况，建议：

### 3.2 座席状态管理

#### 什么是座席状态？

座席状态就像是您在工作中挂的一块"牌子"，告诉系统您现在是否可以接电话。

打个比方：就像超市收银台的"开放/暂停服务"牌子一样，当您的状态是"空闲"时，系统就会把来电分配给您；当您的状态是"通话中"时，系统知道您正忙，就不会再分配新的来电给您。

#### 5种状态的含义和区别

| 状态名称 | 图标颜色 | 含义 | 系统行为 |
|----------|----------|------|----------|
| **空闲** | 🟢 绿色 | 您已准备好，可以接听来电 | 系统会将来电分配给您 |
| **通话中** | 🔴 红色 | 您正在与客户通话 | 系统不会分配新来电（自动切换） |
| **整理** | 🟡 黄色 | 通话刚结束，您正在填写通话小结 | 系统暂时不分配新来电，给您留出整理时间 |
| **休息** | 🟠 橙色 | 您暂时离开座位（如上厕所、喝水） | 系统不会分配来电给您 |
| **离线** | ⚫ 灰色 | 您已下线，不参与来电分配 | 系统完全不分配来电给您 |

⚠️ **注意：** "通话中"状态是系统自动切换的，您不需要手动设置。其他状态需要您手动切换。

#### 如何切换状态

**方法一：通过底部状态栏切换**

1. 找到页面**底部状态栏**左侧的状态显示区域（显示您当前状态的彩色圆点和文字）。
2. 用鼠标点击这个状态区域。
3. 会弹出一个状态选择菜单，显示所有可选状态。
4. 点击您想要切换到的状态。
5. 状态图标和文字会立即更新，表示切换成功。

**方法二：通过软电话面板切换**

1. 点击左侧菜单中的**"软电话"**图标，打开软电话面板。
2. 在软电话面板顶部，找到状态切换下拉框。
3. 点击下拉框，选择您想要的状态。

#### 什么时候应该切换到什么状态

| 场景 | 应该切换到的状态 | 原因 |
|------|-----------------|------|
| 上班开始，准备接电话 | **空闲** | 告诉系统您已准备好，可以接来电 |
| 需要上厕所或短暂离开（5分钟内） | **休息** | 避免来电分配给您但无人接听 |
| 需要较长时间处理其他事务（如开会） | **休息** 或 **离线** | 根据时间长短决定 |
| 通话结束，需要填写小结 | **整理**（通常自动切换） | 给自己留出时间整理记录 |
| 下班或长时间不接电话 | **离线** | 完全退出来电分配队列 |

#### 状态切换的注意事项

⚠️ **注意：**
1. **通话中不能手动切换状态**，必须等通话结束后才能切换。
2. **整理状态有时间限制**，通常为 3-5 分钟（具体时间由管理员设置）。超时后系统会自动将您切换回空闲状态。
3. **休息状态也可能有时间限制**，如果休息时间过长，主管可能会收到提醒。
4. 每次切换状态，系统都会记录切换时间，这些数据会用于统计您的工作效率。
5. 下班前，请务必将状态切换为**"离线"**，再退出系统。

### 3.3 软电话使用

#### 什么是软电话？

软电话（Softphone）就是安装在电脑上的"虚拟电话"。它不是一个实体的电话机，而是一个软件程序，通过您的电脑麦克风和耳机来实现打电话的功能。

打个比方：就像微信语音通话一样，不需要实体电话，用电脑就能打电话。软电话的原理类似，只是它使用的是专业的电话网络，通话质量更稳定。

#### 首次使用前的准备——麦克风权限设置

首次使用软电话时，浏览器会询问是否允许使用麦克风，**必须允许**，否则无法通话。

**Chrome 浏览器麦克风权限设置步骤：**

1. 打开系统后，浏览器地址栏左侧会出现一个**锁形图标**（🔒）或**摄像头/麦克风图标**。
2. 点击该图标，会弹出一个小窗口，显示当前网站的权限设置。
3. 找到"麦克风"选项，确认其状态为**"允许"**。
4. 如果显示"已阻止"，点击旁边的下拉箭头，选择**"允许"**。
5. 页面会提示需要刷新，点击**"重新加载"**按钮。
6. 刷新后，麦克风权限即已开启。

**如果弹出权限请求对话框：**

当您第一次使用软电话时，浏览器顶部可能会弹出一个提示框，询问：
> "callcenter.yourcompany.com 想要使用您的麦克风"

请点击**"允许"**按钮。

⚠️ **注意：** 如果您点击了"阻止"，需要按照上面的步骤手动开启麦克风权限。

#### 软电话界面各部分说明

点击左侧菜单中的"软电话"图标，会打开软电话面板。软电话面板通常显示在页面右侧或以浮动窗口形式出现，包含以下部分：

| 区域 | 说明 |
|------|------|
| 状态显示区 | 显示软电话当前连接状态（已连接/连接中/未连接） |
| 号码显示屏 | 显示您正在拨打或来电的电话号码 |
| 数字键盘 | 0-9 数字键、* 键、# 键，用于输入电话号码 |
| 删除键（⌫） | 删除号码显示屏中最后一个数字 |
| 清空键 | 清空号码显示屏中的所有数字 |
| 拨打按钮（绿色电话图标 📞） | 点击发起呼叫 |
| 挂断按钮（红色电话图标 📵） | 点击结束通话 |
| 接听按钮（绿色） | 来电时点击接听 |
| 拒接按钮（红色） | 来电时点击拒绝接听 |
| 静音按钮（🎤 with ×） | 点击开启/关闭麦克风静音 |
| 保持按钮（⏸） | 点击将通话置于保持状态 |
| 音量调节 | 调节耳机音量大小 |

#### 如何测试麦克风是否正常

在正式接听电话前，建议先测试麦克风是否工作正常：

1. 在软电话面板中，找到**"设备测试"**或**"音频测试"**按钮（通常是一个齿轮图标 ⚙️）。
2. 点击该按钮，进入设备测试页面。
3. 点击**"测试麦克风"**按钮，对着麦克风说话。
4. 如果麦克风正常，页面上的音量指示条会随着您说话而上下波动（像声波一样）。
5. 点击**"测试扬声器"**按钮，系统会播放一段测试音频，确认您能听到声音。
6. 测试完成后，点击**"完成"**或**"关闭"**按钮。

⚠️ **注意：** 如果麦克风测试时音量指示条没有任何反应，请检查：
- 耳机是否已正确插入电脑
- 浏览器麦克风权限是否已开启（参考上面的步骤）
- 电脑系统设置中麦克风是否已启用（Windows：右键点击任务栏音量图标→声音设置→输入设备）

### 3.4 接听来电

接听来电是座席最核心、最重要的工作。下面我们详细介绍整个接听流程。

#### 来电时屏幕会发生什么变化

当有客户来电时，系统会同时发出多种提醒，确保您不会错过来电：

1. **声音提醒：** 耳机中会响起来电铃声（类似手机来电铃声）。
2. **视觉提醒：** 屏幕上会弹出一个来电通知窗口（弹屏）。
3. **页面标题变化：** 浏览器标签页的标题会变成"【来电】XXXXXXXXXX"（显示来电号码）。
4. **底部状态栏变化：** 底部状态栏会显示来电提示，并开始闪烁。
5. **软电话面板变化：** 如果软电话面板是打开的，会显示来电号码和接听/拒接按钮。

#### 来电弹窗说明

来电弹窗通常出现在屏幕的右下角或页面中央，包含以下信息：

| 信息项 | 说明 |
|--------|------|
| 来电号码 | 客户的电话号码（如：138XXXX8888） |
| 客户姓名 | 如果系统中有该号码的客户记录，会显示客户姓名 |
| 客户等级 | 如 VIP、普通客户等（如果有记录） |
| 历史来电次数 | 该客户之前来电的次数 |
| 上次来电时间 | 该客户上次来电的日期和时间 |
| 上次来电原因 | 上次通话的简要记录（如果有填写通话小结） |
| 技能组 | 本次来电所属的技能组（如：售后服务组） |
| 接听按钮 | 绿色按钮，点击接听来电 |
| 拒接按钮 | 红色按钮，点击拒绝接听（一般情况下不应拒接） |

💡 **提示：** 来电弹窗显示的客户信息可以帮助您在接听前就了解客户情况，做好准备。例如，如果看到客户是 VIP，可以提前准备更周到的服务。

#### 如何接听来电

**方法一：点击来电弹窗上的接听按钮**

1. 来电弹窗出现后，找到绿色的**"接听"**按钮（通常显示为绿色电话图标 📞 或"接听"文字）。
2. 用鼠标点击该按钮。
3. 系统会立即接通电话，您可以开始与客户通话。

**方法二：点击软电话面板上的接听按钮**

1. 如果软电话面板是打开的，来电时面板上会显示来电号码和接听按钮。
2. 点击软电话面板上的绿色接听按钮。

**方法三：使用键盘快捷键（如果系统支持）**

- 部分系统支持按 **F8** 键或其他快捷键接听来电，具体请询问您的管理员。

#### 接听后界面变化

成功接听来电后，界面会发生以下变化：

1. **来电弹窗消失**，替换为通话中界面。
2. **底部状态栏**显示"通话中"（红色），并开始计时（00:00:01, 00:00:02...）。
3. **中央工作区**自动切换到客户信息页面，显示该客户的详细资料。
4. **软电话面板**显示通话控制按钮（静音、保持、转接、挂断等）。
5. **您的座席状态**自动切换为"通话中"（红色）。

#### 接听后如何与客户通话

接听成功后，您就可以直接通过耳机与客户通话了：

1. 确认耳机已正确佩戴，麦克风位置靠近嘴巴（距离约 2-3 厘米）。
2. 用正常说话的音量与客户交流，不需要大声喊叫。
3. 通话过程中，您可以同时在电脑上查看客户信息、记录要点。

💡 **提示：** 接听电话时的标准开场白示例：
> "您好，这里是[公司名称]客服中心，我是[您的姓名]，请问有什么可以帮助您？"

#### 通话中界面说明

通话进行中时，界面上会显示以下控制按钮：

| 按钮名称 | 图标 | 功能说明 |
|----------|------|----------|
| 静音 | 🎤 | 关闭/开启您的麦克风，客户听不到您说话 |
| 保持 | ⏸ | 将客户置于等待状态，客户会听到等待音乐 |
| 转接 | ↗ | 将通话转给其他座席或部门 |
| 三方 | 👥 | 邀请第三方加入通话 |
| 录音 | ⏺ | 手动开始/停止录音（通常系统自动录音） |
| 挂断 | 📵 | 结束通话 |
| 数字键盘 | 🔢 | 打开数字键盘，用于按键操作（如输入分机号） |

通话时长显示在界面顶部或底部状态栏，格式为 `00:00:00`（时:分:秒）。

#### 如何查看客户信息

通话中，中央工作区会自动显示客户信息，包括：

**基本信息标签页：**
- 客户姓名
- 联系电话（可能有多个）
- 客户等级（普通/VIP/黑名单等）
- 所在地区
- 注册时间

**历史记录标签页：**
- 历史来电记录（时间、时长、处理结果）
- 历史工单记录（工单编号、问题描述、处理状态）
- 历史备注信息

**订单信息标签页（如果系统与订单系统对接）：**
- 客户的订单列表
- 订单状态

💡 **提示：** 通话中可以点击不同的标签页切换查看不同类型的客户信息，这样可以更全面地了解客户情况，提供更好的服务。

#### 如何在通话中记录信息

通话过程中，您可以随时记录重要信息：

**方法一：使用右侧快捷备注框**

1. 在右侧信息面板中，找到**"快捷备注"**文本框。
2. 直接点击文本框，输入您想记录的内容。
3. 这些内容会在通话结束后自动带入通话小结。

**方法二：在客户信息页面添加备注**

1. 在客户信息页面，找到**"备注"**字段。
2. 点击备注字段旁边的**"编辑"**按钮（铅笔图标 ✏️）。
3. 输入备注内容，点击**"保存"**。

⚠️ **注意：** 通话中记录信息时，请注意不要因为打字而忽略了与客户的对话，保持与客户的沟通是最重要的。

### 3.5 主动拨打电话（呼出）

#### 什么情况下需要主动拨打电话

以下情况需要您主动给客户打电话：

- 客户之前来电但未能接通，需要回拨
- 主管安排的外呼任务（如回访、通知等）
- 工单处理过程中需要联系客户确认信息
- 客户留言要求回电

#### 如何输入电话号码

**方式一：使用软电话数字键盘输入**

1. 点击左侧菜单中的**"软电话"**图标，打开软电话面板。
2. 确认软电话状态为**"已连接"**（绿色指示灯）。
3. 在软电话面板的数字键盘上，依次点击电话号码的每一位数字。
4. 号码显示屏中会实时显示您输入的号码。
5. 如果输入错误，点击**删除键（⌫）**删除最后一位，或点击**清空键**清除全部。

**方式二：直接用键盘输入**

1. 打开软电话面板。
2. 点击号码显示屏（输入框），使其获得焦点（出现光标）。
3. 直接用电脑键盘输入电话号码。
4. 号码会实时显示在号码显示屏中。

**方式三：从客户信息页面直接拨打**

1. 在客户管理页面找到目标客户。
2. 在客户信息中，找到电话号码旁边的**"拨打"**按钮（电话图标 📞）。
3. 点击该按钮，系统会自动将号码填入软电话并发起呼叫。

💡 **提示：** 拨打手机号码时，直接输入11位手机号即可。拨打固定电话时，通常需要加区号（如北京：010-XXXXXXXX）。

#### 如何发起呼叫

1. 确认号码显示屏中显示的号码正确无误。
2. 点击软电话面板上的**绿色拨打按钮**（📞）。
3. 或者，在号码输入完成后，按键盘上的 **Enter（回车）** 键。
4. 系统开始拨号，您会听到拨号音。

#### 等待接听时的界面

发起呼叫后，在对方接听之前：

- 软电话面板显示"呼叫中..."或"等待接听..."
- 号码显示屏显示正在拨打的号码
- 您的耳机中会听到回铃音（"嘟...嘟...嘟..."）
- 底部状态栏显示"呼叫中"状态
- 界面上会显示**"取消呼叫"**按钮，点击可以取消本次呼叫

#### 对方接听后的操作

当对方接听电话后：

1. 回铃音停止，通话开始计时。
2. 界面切换为通话中状态，显示通话控制按钮。
3. 您可以开始与对方通话。
4. 通话中的操作与接听来电后相同（参考 3.4 节）。

#### 对方未接听怎么办

如果拨出后对方长时间未接听（通常等待 30-60 秒后自动挂断），或者对方主动拒接：

1. 系统会自动结束本次呼叫。
2. 界面恢复到通话前的状态。
3. 您的状态会切换回"空闲"或"整理"。

**处理建议：**

| 情况 | 建议处理方式 |
|------|-------------|
| 对方未接听 | 稍后再试，或发送短信通知（如果系统支持） |
| 对方拒接 | 记录在工单备注中，等待客户主动来电 |
| 号码错误（提示空号） | 核实客户号码，联系客户确认正确号码 |
| 号码停机 | 记录在工单中，通过其他方式联系客户 |

⚠️ **注意：** 拨出电话时，请确认您有权限拨打该号码。部分系统会限制可拨打的号码范围（如只能拨打国内号码）。

### 3.6 通话中的各种操作

在通话过程中，除了正常交流外，您还可以使用多种功能来更好地服务客户。

#### 保持通话

**什么是保持？**

"保持"就是暂时让客户等一等。当您按下保持按钮后，客户会听到一段等待音乐（而不是沉默），而您可以暂时离开通话去查资料、请示主管或处理其他事情。

**什么时候使用保持？**

- 需要查询系统中的信息，需要一点时间
- 需要请示主管或同事
- 需要联系其他部门确认信息
- 需要短暂处理其他紧急事务

**如何操作保持：**

1. 通话中，找到软电话面板或通话控制区的**"保持"**按钮（⏸ 图标）。
2. 点击**"保持"**按钮。
3. 客户端会自动播放等待音乐，客户听不到您的声音。
4. 界面上的保持按钮会变为激活状态（通常变色或显示"取消保持"）。
5. 处理完毕后，再次点击**"保持"**按钮（或"取消保持"按钮），恢复通话。
6. 恢复通话后，告知客户："感谢您的耐心等待，我已经为您查询到了..."

⚠️ **注意：**
- 保持时间不宜过长，建议不超过 **2 分钟**。
- 如果需要较长时间，应先告知客户预计等待时间，并征得客户同意。
- 保持前请告知客户："请您稍等，我帮您查询一下，大约需要 X 分钟。"

---

#### 静音

**什么是静音？**

静音是关闭您的麦克风，这样客户听不到您说话，但您仍然可以听到客户说话。与"保持"不同，静音时客户不会听到等待音乐，而是听到正常的通话环境（可能有背景噪音）。

**什么时候使用静音？**

- 需要咳嗽、打喷嚏时，避免影响客户
- 需要与旁边同事短暂交流时
- 背景噪音较大时，暂时关闭麦克风

**如何操作静音：**

1. 通话中，找到软电话面板的**"静音"**按钮（🎤 带删除线的图标）。
2. 点击**"静音"**按钮。
3. 麦克风图标会变为红色或带删除线，表示静音已开启。
4. 此时您说话客户听不到，但您仍能听到客户。
5. 需要说话时，再次点击**"静音"**按钮取消静音。

⚠️ **注意：** 静音后请记得及时取消静音，否则客户说话您能听到，但您说话客户听不到，会造成误解。

---

#### 转接电话

**什么是转接？**

转接就是把您正在接听的电话"传"给另一个座席或部门。就像在公司里，前台接到电话后转给相关部门一样。

**什么时候需要转接？**

- 客户的问题不在您的处理范围内，需要转给专业部门
- 客户指定要找某位同事
- 需要将投诉类电话转给投诉处理专员

**转接的两种方式：**

**方式一：盲转（直接转接，不通知对方）**

1. 通话中，点击**"转接"**按钮（↗ 图标）。
2. 弹出转接面板，显示可转接的座席列表或输入框。
3. 在搜索框中输入目标座席的姓名或工号，或从列表中选择。
4. 点击目标座席旁边的**"转接"**按钮。
5. 系统立即将通话转给目标座席，您的通话自动结束。
6. 目标座席的电话会响起，接听后即可与客户通话。

**方式二：咨询转接（先与对方确认，再转接）**

1. 通话中，点击**"转接"**按钮。
2. 选择目标座席，点击**"咨询"**按钮（而不是直接转接）。
3. 系统会先将客户置于保持状态，然后拨通目标座席的电话。
4. 您与目标座席通话，说明情况，确认对方可以接手。
5. 确认后，点击**"完成转接"**按钮，通话正式转给目标座席。
6. 或者，如果对方无法接手，点击**"取消转接"**，恢复与客户的通话。

⚠️ **注意：**
- 转接前，请先告知客户："我现在帮您转接到[部门/同事]，请稍等。"
- 确认目标座席处于空闲状态再转接，避免转接失败。
- 转接后，您的通话结束，需要填写通话小结。

---

#### 三方通话

**什么是三方通话？**

三方通话就是三个人同时在一个电话里通话。例如：您（座席）、客户、专家三个人同时通话，专家可以直接回答客户的专业问题。

**什么时候使用三方通话？**

- 需要专家或主管直接与客户沟通
- 需要多方确认某件事情
- 客户需要与另一个部门的人直接沟通

**如何操作三方通话：**

1. 通话中，点击**"三方"**或**"邀请"**按钮（👥 图标）。
2. 弹出邀请面板，输入要邀请的第三方号码或从座席列表中选择。
3. 点击**"邀请"**按钮，系统开始拨打第三方。
4. 此时客户会被置于保持状态，您与第三方通话。
5. 第三方接听后，点击**"合并通话"**按钮，三方通话正式开始。
6. 三方通话中，三个人都能互相听到对方说话。
7. 任何一方挂断，剩余两方仍可继续通话。

⚠️ **注意：** 三方通话会消耗更多的网络带宽，如果网络不稳定，可能影响通话质量。

---

#### 挂断电话

**如何挂断：**

1. 通话结束后，点击软电话面板上的**红色挂断按钮**（📵 图标）。
2. 或者，点击通话控制区的**"挂断"**按钮。
3. 通话立即结束，计时停止。

**挂断后会发生什么：**

1. 通话结束，耳机中的通话声音消失。
2. 系统自动将您的状态切换为**"整理"**（黄色）。
3. 屏幕上弹出**"通话小结"**填写窗口（参考 3.7 节）。
4. 整理状态有倒计时，请在规定时间内完成小结填写。

⚠️ **注意：**
- 请等客户先挂断，或者在确认客户问题已解决后再挂断，不要在客户还在说话时突然挂断。
- 挂断前的标准结束语示例："感谢您的来电，如有其他问题欢迎再次联系，再见！"

---

### 3.7 通话结束后的操作

#### 通话小结填写

**为什么要填写通话小结？**

通话小结是对本次通话内容的简短记录，就像工作日记一样。填写通话小结有以下好处：

- 方便下次接到同一客户来电时，快速了解历史情况
- 帮助主管了解座席的工作内容
- 为质检提供参考
- 积累客户服务数据，帮助公司改进服务

**填什么内容？**

通话小结通常包含以下字段：

| 字段名称 | 是否必填 | 填写说明 |
|----------|----------|----------|
| 通话类型 | 必填 | 从下拉菜单选择：咨询、投诉、建议、报修、其他 |
| 问题分类 | 必填 | 从下拉菜单选择客户问题的具体类别 |
| 处理结果 | 必填 | 从下拉菜单选择：已解决、转工单、转接、无法解决、其他 |
| 通话摘要 | 建议填写 | 用简短的文字描述本次通话的主要内容（50-200字） |
| 关联工单 | 选填 | 如果本次通话创建了工单，填写工单编号 |
| 备注 | 选填 | 其他需要记录的信息 |

**如何填写通话小结：**

1. 通话挂断后，系统自动弹出通话小结填写窗口。
2. 点击**"通话类型"**下拉框，选择本次通话的类型。
3. 点击**"问题分类"**下拉框，选择客户问题的分类。
4. 点击**"处理结果"**下拉框，选择本次通话的处理结果。
5. 在**"通话摘要"**文本框中，输入本次通话的简要描述。
6. 如果需要创建工单，点击**"创建工单"**按钮（参考 3.9 节）。
7. 填写完成后，点击**"提交"**按钮保存小结。
8. 提交成功后，系统会提示"小结已保存"，并自动将您的状态切换为"空闲"。

⚠️ **注意：**
- 整理状态有时间限制（通常 3-5 分钟），请在时间内完成填写。
- 如果时间到了还没填完，系统会自动将状态切换为"空闲"，但小结仍可以在通话记录中补填。
- 通话摘要请用简洁的语言描述，不需要逐字记录对话内容。

💡 **提示：** 通话摘要的写法示例：
> "客户咨询产品A的使用方法，已向客户详细说明操作步骤，客户表示已了解，问题已解决。"

---

#### 满意度评价说明

通话结束后，系统可能会自动向客户发送满意度评价邀请：

- **短信方式：** 系统自动向客户手机发送一条短信，邀请客户对本次服务打分。
- **IVR方式：** 挂断后，客户端会自动播放语音提示，邀请客户按键打分（如：对本次服务满意请按1，不满意请按2）。

作为座席，您不需要做任何操作，系统会自动处理。但请注意：

- 您的服务质量会直接影响客户的满意度评分。
- 满意度数据会被统计，作为您绩效考核的参考指标之一。

---

#### 如何切换回空闲状态

填写完通话小结并提交后，系统通常会自动将您的状态切换为"空闲"。如果没有自动切换，请手动操作：

1. 找到底部状态栏的状态显示区域。
2. 点击当前状态（"整理"）。
3. 在弹出的菜单中，选择**"空闲"**。
4. 状态图标变为绿色，表示您已准备好接听下一个来电。

---

### 3.8 客户信息管理

#### 如何查看客户信息

**方法一：通话中自动显示**

来电时，系统会自动在中央工作区显示该客户的信息（如果系统中有该号码的记录）。

**方法二：通过客户管理菜单查看**

1. 点击左侧菜单中的**"客户管理"**图标。
2. 进入客户列表页面，显示所有客户的列表。

### 3.9 工单管理

#### 什么是工单？

工单就像是一张"任务单"或"问题跟踪单"。

打个比方：当您去维修店修手机，店员会给您开一张维修单，上面写着您的手机型号、问题描述、预计完成时间等。这张维修单就是"工单"。

在呼叫中心，当客户的问题不能在一次通话中立即解决（比如需要后台处理、需要其他部门配合），就需要创建一个工单，把问题记录下来，跟踪处理进度，直到问题最终解决。

**工单的作用：**
- 确保客户的问题不会被遗忘
- 跟踪问题的处理进度
- 多人协作处理同一个问题
- 留存处理记录，方便查询

---

#### 什么情况下需要创建工单

| 情况 | 是否需要创建工单 |
|------|-----------------|
| 客户问题当场解决 | 不需要（填写通话小结即可） |
| 客户问题需要后台处理（如退款、补发等） | 需要 |
| 客户投诉，需要调查处理 | 需要 |
| 客户报修，需要安排上门服务 | 需要 |
| 客户问题需要其他部门处理 | 需要 |
| 客户要求回访或跟进 | 需要 |

---

#### 如何创建工单

**方法一：通话中创建**

1. 通话中，在右侧信息面板或通话控制区，找到**"创建工单"**按钮。
2. 点击该按钮，弹出工单创建表单。
3. 填写工单信息（详见下方字段说明）。
4. 点击**"提交"**按钮，工单创建成功。

**方法二：通话小结中创建**

1. 通话结束后，在填写通话小结时，点击**"创建工单"**按钮。
2. 弹出工单创建表单，填写信息后提交。

**方法三：在工单管理页面创建**

1. 点击左侧菜单中的**"工单管理"**。
2. 点击右上角的**"新建工单"**按钮（蓝色"+ 新建"按钮）。
3. 填写工单信息后提交。

**工单字段详细说明：**

| 字段名称 | 是否必填 | 说明 |
|----------|----------|------|
| 工单标题 | 必填 | 简短描述问题，如"客户申请退款" |
| 工单类型 | 必填 | 从下拉菜单选择：投诉、咨询、报修、建议、其他 |
| 优先级 | 必填 | 紧急/高/中/低，根据问题紧急程度选择 |
| 关联客户 | 必填 | 搜索并选择相关客户 |
| 问题描述 | 必填 | 详细描述客户的问题，越详细越好 |
| 处理要求 | 选填 | 说明期望的处理方式或结果 |
| 截止时间 | 选填 | 客户要求的处理截止时间 |
| 指派给 | 选填 | 将工单指派给特定的处理人员或部门 |
| 附件 | 选填 | 上传相关截图、文件等 |
| 备注 | 选填 | 其他补充信息 |

**填写工单的注意事项：**

⚠️ **注意：**
- 问题描述要尽量详细，包括：客户反映的具体问题、发生时间、影响范围等。
- 优先级要根据实际情况选择，不要所有工单都选"紧急"。
- 如果有截图或相关文件，请上传作为附件，方便处理人员了解情况。

---

#### 工单编号是什么，在哪里看

工单编号是系统自动生成的唯一标识符，用于区分不同的工单。

- **格式示例：** `WD20240115001`（WD=工单，20240115=日期，001=当日序号）
- **在哪里看：** 工单创建成功后，系统会显示工单编号。在工单列表和工单详情页面的顶部也会显示工单编号。
- **用途：** 客户询问处理进度时，可以报工单编号；与同事沟通时，用工单编号指代具体工单。

💡 **提示：** 创建工单后，建议将工单编号告知客户，方便客户后续查询处理进度。

---

#### 如何查看自己的工单

1. 点击左侧菜单中的**"工单管理"**。
2. 进入工单列表页面，默认显示所有与您相关的工单。
3. 可以通过以下方式筛选工单：
   - 点击**"我创建的"**标签，查看您创建的工单
   - 点击**"指派给我的"**标签，查看需要您处理的工单
   - 使用顶部的筛选条件（状态、时间范围、工单类型等）

**工单列表显示的信息：**

| 列名 | 说明 |
|------|------|
| 工单编号 | 工单的唯一标识 |
| 工单标题 | 工单的简短描述 |
| 客户姓名 | 关联的客户 |
| 工单类型 | 投诉/咨询/报修等 |
| 优先级 | 紧急/高/中/低 |
| 状态 | 当前处理状态 |
| 创建时间 | 工单创建的时间 |
| 截止时间 | 要求处理完成的时间 |
| 处理人 | 当前负责处理的人员 |
| 操作 | 查看、处理、关闭等操作按钮 |

---

#### 如何处理工单

1. 在工单列表中，找到需要处理的工单。
2. 点击工单标题或行末的**"处理"**按钮，进入工单详情页面。
3. 查看工单的详细信息，了解问题背景。
4. 在**"处理记录"**区域，点击**"添加处理记录"**按钮。
5. 在弹出的文本框中，输入本次处理的内容和结果。
6. 选择处理后的工单状态（处理中/已解决等）。
7. 点击**"提交"**按钮，保存处理记录。

---

#### 工单状态说明

| 状态 | 含义 | 说明 |
|------|------|------|
| **待处理** | 工单刚创建，还没有人开始处理 | 需要尽快指派处理人员 |
| **处理中** | 已有人员在处理，但尚未完成 | 处理人员正在跟进 |
| **已解决** | 问题已经处理完毕，等待客户确认 | 可以联系客户确认是否满意 |
| **已关闭** | 工单已完全结束，不再跟进 | 问题已解决且客户确认，或工单已作废 |

---

#### 如何关闭工单

当客户的问题已经完全解决后，需要关闭工单：

1. 进入工单详情页面。
2. 确认问题已解决，客户已满意。
3. 点击页面右上角的**"关闭工单"**按钮。
4. 弹出确认对话框，填写关闭原因（如："问题已解决，客户确认满意"）。
5. 点击**"确认关闭"**按钮。
6. 工单状态变为"已关闭"，不可再修改。

⚠️ **注意：**
- 工单关闭后无法重新打开，如果问题再次出现，需要创建新工单。
- 关闭工单前，请确认问题确实已经解决，不要随意关闭未解决的工单。

## 1. 实时监控面板

### 访问方式

登录管理后台后，默认进入实时监控面板。

### 关键指标说明

| 指标 | 说明 | 预警阈值 |
|------|------|---------|
| 在线座席 | 当前已登录的座席总数 | - |
| 通话中 | 正在处理呼叫的座席数 | - |
| 空闲座席 | 可接收新呼叫的座席数 | < 3 时建议关注 |
| 队列等待 | 当前排队等待的呼叫数 | > 10 时红色预警 |
| 接通率 | 今日呼叫接通率 | < 85% 时需关注 |
| 平均等待 | 今日平均等待时长（秒） | > 60 秒需关注 |

### 大屏展示模式

点击右上角「全屏」按钮，切换为大屏展示模式，适合在监控大屏上展示。

### 座席实时状态表

表格实时显示每位座席的：
- 当前状态（颜色标注）
- 状态持续时长
- 今日接听量
- 平均处理时长

> 💡 **提示：** 数据每 5 秒自动刷新一次。

## 2. 座席管理

### 添加座席

1. 进入「座席管理」→「座席列表」
2. 点击「新增座席」
3. 填写信息：
   - 工号（唯一，如 A001）
   - 关联用户账号
   - 分机号
   - 所属技能组
   - 座席级别（普通/高级）
4. 点击「保存」

### 编辑座席信息

1. 在座席列表中找到目标座席
2. 点击「编辑」按钮
3. 修改信息后保存

### 启用/禁用座席

- 点击座席行末尾的「启用/禁用」开关
- 禁用后该座席无法登录系统

### 培训模式管理

1. 找到目标座席，点击「培训模式」
2. 开启后，该座席不会收到真实客户呼叫
3. 主管可实时监听培训座席的通话
4. 培训完成后关闭培训模式

## 3. 技能组管理

### 创建技能组

1. 进入「座席管理」→「技能组」
2. 点击「新建技能组」
3. 填写：
   - 技能组代码（如 TECH_SUPPORT）
   - 技能组名称
   - 描述
4. 保存后，在座席管理中将座席分配到该技能组

### 技能组路由规则

IVR 菜单中的「转人工」节点可指定目标技能组，系统将呼叫路由到对应技能组的空闲座席。

## 4. 客户管理

### 搜索客户

支持按以下条件搜索：
- 电话号码（精确/模糊）
- 客户姓名（模糊）
- VIP 等级

### 设置 VIP 等级

1. 找到目标客户，点击「编辑」
2. 修改 VIP 等级（普通/VIP/SVIP）
3. 保存后，该客户来电时将优先分配给高级座席

### 查看客户历史

在客户详情页可查看：
- 所有历史通话记录
- 所有历史工单
- 满意度评分趋势

## 5. IVR 配置

### IVR 流程编辑器

进入「系统配置」→「IVR 管理」，使用可视化流程编辑器配置 IVR。

### 节点类型说明

| 节点类型 | 说明 |
|---------|------|
| 播放语音 | 播放指定的语音文件 |
| 收集按键 | 等待客户按键，根据按键路由 |
| 转人工 | 将呼叫转入指定技能组队列 |
| 播放 TTS | 将文字转为语音播放 |
| 挂断 | 结束呼叫 |

### 上传语音文件

1. 进入「IVR 管理」→「语音文件」
2. 点击「上传」，选择 WAV 或 MP3 文件
3. 上传后可在 IVR 节点中引用

### 超时处理规则

- 客户 **10 秒**内未按键：重复当前菜单
- 连续 **3 次**未按键：自动转人工服务

## 6. 质检管理

### 创建质检模板

1. 进入「质检管理」→「评分模板」
2. 点击「新建模板」
3. 添加评分项（如服务态度、业务能力等）
4. 设置每项满分和通过分数线（默认 60 分）

### 分配质检任务

1. 进入「质检管理」→「质检任务」
2. 选择需要质检的通话记录
3. 分配给质检员

### 查看质检结果

- 质检完成后，结果关联到通话记录
- 评分低于 60 分自动标记为不合格
- 系统自动通知座席主管

## 7. 报表中心

### 生成报表

1. 进入「报表中心」
2. 选择报表类型：
   - 座席绩效报表
   - 呼叫统计报表
   - 客户满意度报表
   - 工单统计报表
3. 设置时间范围和筛选条件
4. 点击「生成报表」（异步处理，约 30 秒内完成）

### 导出报表

报表生成完成后：
- 点击「导出 Excel」下载 Excel 格式
- 点击「导出 PDF」下载 PDF 格式

### 定时报表

1. 进入「报表中心」→「定时报表」
2. 配置报表类型、发送时间和接收邮箱
3. 系统每天在指定时间自动生成并发送

## 8. 权限管理

### 用户管理

1. 进入「系统管理」→「用户管理」
2. 可进行以下操作：
   - 新增用户
   - 修改用户角色
   - 重置密码
   - 启用/禁用账号

### 角色说明

| 角色 | 权限范围 |
|------|---------|
| 管理员 | 所有功能和数据 |
| 座席主管 | 本组座席管理、本组数据查看、质检 |
| 座席 | 自己的通话记录和工单 |
| 质检员 | 通话记录查看、质检评分 |

### API 密钥管理

用于第三方系统集成：
1. 进入「系统管理」→「API 密钥」
2. 点击「生成密钥」
3. 设置密钥名称和限流规则（默认 100 次/分钟）
4. 将密钥提供给第三方系统

## 9. 系统配置

### 常用配置项

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 队列最大等待时长 | 超时后播放道歉语音 | 300 秒 |
| 座席整理超时 | 超时后发送提醒 | 180 秒 |
| 录音保留天数 | 录音文件保留时间 | 90 天 |
| 满意度评价超时 | 客户未评价的超时时间 | 15 秒 |
| 工单超时提醒 | 未处理工单的提醒时间 | 24 小时 |
| 队列预警阈值 | 队列超过此数量时预警 | 10 个 |

### 修改配置

1. 进入「系统管理」→「系统配置」
2. 找到目标配置项，点击「编辑」
3. 修改值后保存
4. 配置在 **10 秒内**生效，无需重启服务

## 10. 黑名单管理

### 添加黑名单

1. 进入「客户管理」→「黑名单」
2. 点击「添加」
3. 输入电话号码和备注原因
4. 保存后立即生效，该号码呼入将被自动拒接

### 移除黑名单

1. 在黑名单列表中找到目标号码
2. 点击「移除」
3. 确认后立即生效，该号码可正常呼入

### 查询黑名单

支持按电话号码或备注内容搜索。

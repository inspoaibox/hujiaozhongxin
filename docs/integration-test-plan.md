# 千牛云呼叫中心系统 - 集成联调测试计划
# 作者：深圳市千牛云科技有限公司

## 1. 前后端接口联调（座席工作台）

### 1.1 登录流程
- [ ] POST /api/auth/login - 用户登录，获取 JWT Token
- [ ] WebSocket 连接建立（携带 Token）
- [ ] 座席状态初始化为"空闲"

### 1.2 呼叫流程
- [ ] 呼入呼叫 → 来电弹屏显示客户信息
- [ ] 接听呼叫 → 座席状态变为"通话中"
- [ ] 挂断呼叫 → 座席状态变为"整理"
- [ ] 填写通话小结 → 座席状态变为"空闲"

### 1.3 工单流程
- [ ] 创建工单 → 工单编号自动生成
- [ ] 工单分配 → 被分配人收到通知
- [ ] 工单状态流转 → 历史记录正确

## 2. FreeSWITCH 与呼叫管理服务联调

### 2.1 呼入测试
- [ ] 拨打测试号码 → FreeSWITCH 收到呼叫
- [ ] 触发 CHANNEL_CREATE 事件 → 呼叫记录创建
- [ ] IVR 播放欢迎语音
- [ ] 按键选择人工服务 → 进入队列
- [ ] 座席接听 → CHANNEL_ANSWER 事件触发
- [ ] 录音自动开始

### 2.2 呼出测试
- [ ] 座席发起呼出 → FreeSWITCH 拨号
- [ ] 客户接听 → 通话建立
- [ ] 挂断 → 通话记录保存

### 2.3 转接测试
- [ ] 盲转 → 呼叫转接到目标座席
- [ ] 目标座席未接听 → 呼叫返回原座席

## 3. 端到端呼叫全流程测试

### 场景1：普通呼入
1. 客户拨打 → IVR 欢迎 → 按1人工 → 队列等待 → 座席接听
2. 通话中 → 挂断 → 满意度评价 → 录音保存

### 场景2：VIP 客户呼入
1. VIP 客户拨打 → 识别 VIP → 优先分配高级座席
2. 来电弹屏显示 VIP 标识

### 场景3：队列超时
1. 所有座席忙碌 → 客户等待 300 秒
2. 播放道歉语音 → 提供留言选项
